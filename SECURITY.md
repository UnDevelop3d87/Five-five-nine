## Reporting a Security Vulnerability

The security and privacy of Fresno Resource Hub™ and its visitors —
many of whom are vulnerable community members — is a top priority.

If you discover a security vulnerability, a privacy concern, or any
issue that could put visitors' safety or data at risk, please report
it responsibly.

**Please do NOT open a public issue for security problems.**

Instead, email us directly at:
**fresnoresourcehub@mailfence.com**

Please include:
- A description of the issue
- Steps to reproduce it (if applicable)
- The page or feature affected
- Any suggested fix (optional)

## What to Expect

- We will acknowledge your report as soon as reasonably possible.
- We will investigate and work to address valid issues promptly.
- We ask that you give us reasonable time to fix the issue before
  disclosing it publicly.
- We deeply appreciate responsible disclosure and the effort of
  security researchers in keeping our community safe.

## Scope

This project is a static informational website. It does not store
sensitive personal records on a server. Reports related to data
exposure, insecure links, or anything affecting visitor privacy are
especially welcome.

## Supported Versions

The live version of the site at fivefiveninehub.netlify.app is the
only actively maintained version.
